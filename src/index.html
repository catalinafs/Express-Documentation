Introduction

Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications. It makes much easier to create a server and handle routing for your application, Express handles things like directing people to the correct page when they visit a certain endpoint like /blog.

Installing Express

So that you can install express in your project you have to have installed Node.js , In case you do not have it installed, please go to the page of Node.js and download it, once this step is done we can continue with the installation of express.

Create a folder for your app and make it your current working folder.

mkdir your-name-app
cd your-name-app

Then use the command npm init to create a package.json file for your application. For more information on how a package.json works, check out npm's package.json handling

npm init

This command asks you a series of things in order to configure your package.json. For now, you can just click ENTER with the following exception:

entry point: (index.js)

Enter app.js, or the name you want the main file to have.

Now you can install Express.js using the following command:

npm install express or npm i express

If you want to temporarily install Express.js use the following command:

npm install express --no-save or npm i express --no-save
